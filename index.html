Tiny Code Sample (JavaScript)

The output shows the particle’s exact position on the circle at the current moment, using its updated angle.


// click "Run snippet"
function lerp(a, b, t) {
return a + (b - a) * t;
}

const particle = { radius: 80, angle: 0 };
const settings = { drift: 0.015 };

particle.angle += settings.drift;

const x = Math.cos(particle.angle) * particle.radius;
const y = Math.sin(particle.angle) * particle.radius;

`particle at (${x.toFixed(2)}, ${y.toFixed(2)})`;

About Me.

I am a data scientist and researcher with a background in biological data science, neuroimaging, and clinical analytics. My current work focuses on multimodal machine learning ranging from imaging to NLP systems that analyze real-world health signals. Alongside my research, I am active as a 2D digital artist, which informs my interest in visual representation, UI/UX design, and the interpretability of scientific communication. Currently, I am pursuing a Master's degree in health data science at UCSF while taking a lead position at BOLERO Inc., a full stack music management company, as data scientist and developer.
